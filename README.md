TakeTwo

Overview
TakeTwo is a film talent networking and audition platform that connects actors, directors, and creative professionals. It enables users to build portfolios, discover talent, post and apply for auditions, and collaborate on film projects through a modern, cinematic interface.

Live Demo
Repository: https://github.com/cyriljoesaji-lab/taketwo

Features
Multi-role profiles (Actor, Director, Writer, Cinematographer, Editor, Composer, Producer, Enthusiast)
Portfolio with images and video showreels
Talent discovery with filters (location, skills, experience)
Audition posting and application workflow
Messaging for collaboration and file sharing
Project collaboration and team recruitment
Dark and light themes
Smooth transitions and micro-interactions
